Для создания двух проектов Java на 8 версии необходимо выполнить следующие шаги:

Запустите среду разработки IntelliJ IDEA.
Нажмите на кнопку "Create New Project" на стартовом экране или выберите "File" -> "New" -> "Project".
Выберите тип проекта "Java".
Укажите настройки проекта, такие как имя, место на диске, и версию Java SDK. Для проекта на 8 версии выберите JDK 8.
Нажмите "Finish", чтобы создать проект.
Для переключения одного проекта на версию 11 необходимо выполнить следующие шаги:

Откройте проект в IntelliJ IDEA.
Нажмите на "File" -> "Project Structure".
Выберите "Project" в левом меню.
В разделе "Project SDK" выберите версию JDK 11.
Нажмите "Apply" и "OK", чтобы сохранить изменения.
Для переключения другого проекта на версию 17 необходимо выполнить следующие шаги:

Откройте проект в IntelliJ IDEA.
Нажмите на "File" -> "Project Structure".
Выберите "Project" в левом меню.
В разделе "Project SDK" выберите версию JDK 17.
Нажмите "Apply" и "OK", чтобы сохранить изменения.
